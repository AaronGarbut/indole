# FileLogInterface

copy the stream with no change, logging the content to a file at the same time.

This plugin can be use to debug, or boardcast.

## Args

| name     | type   | description       |
| -------- | ------ | ----------------- |
| FileName | string | the log file name |


## IO

| --- | type   | size   |
| --- | ------ | ------ |
| I   | stream | `-`    |
| O   | stream | `-` |
